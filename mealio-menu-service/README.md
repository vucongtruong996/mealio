# Mealio Menu Service

## Responsibility

- Category Management
- Menu Item Management
- Nutrition Information
- Menu Search

## Technology

- Java 17
- Spring Boot 3
- Maven
- PostgreSQL
- Flyway
- Lombok
- Testcontainers

## Development

Build:

```bash
./mvnw clean compile
```

Run tests:

```bash
./mvnw test
```

Run application:

```bash
./mvnw spring-boot:run
```
