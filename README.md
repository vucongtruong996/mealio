# Mealio

Mealio is an AI Agentic Food Ordering and Nutrition Platform.

The project is designed to demonstrate production-level backend engineering using modern Java technologies and AI Agent architecture.

## Core Technologies

- Java 17
- Spring Boot
- PostgreSQL
- Redis
- Docker
- REST API
- gRPC
- AWS
- AI Agents
- MCP
- RAG
- pgvector

## Architecture Principles

- Microservices
- Database per Service
- API First
- TDD
- SOLID
- Clean Architecture
